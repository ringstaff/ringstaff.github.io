// dinolab

Dinolab is a company I created to make imaginative, musical, and educational apps for kids. The first app, which is currently ready for release on Android, is a soothing musical experience for tired kids, called Sleepy Stars. The app opens to show a night sky filled with stars that twinkle in rhythm to soft lullaby music (5 songs with original arrangements). The user can interact by tapping a star in the center, which skips to the next song. The beatiful music, glowing stars, and limited interaction allow the user to become sleepy, by creating a calming for bedtime (set for any age, but is geared for children). Future projects for dinolab include an app that is both parts educational and an engaging game that lets the user be the orchestrator by toggling different musical parts, instruments and rhythm in an orchestra scene (set for ages 2+).
